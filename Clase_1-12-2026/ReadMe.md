el metodo .index() sirve para traer cierta cadena de caracteres, de izquierda a derecha, este nos trae la posicion
de donde se encuentra un caracter o substring

el metodo upper() nos sirve para pasar a mayusculas todos los
caracteres de una cadena.

el metodo lower() es lo inverso que vimos en el uppder(), convierte a todos los carateres a minisculas

El metodo split() corta un string a una lista de string, usando un char como separados

El metodo join() es lo contrario, podemos unir varios string
y crear uno solo importante de usar un char o string " " o
' 'como el que va a unir los string " ".join (lista) o
" ".join(["Hi", "Mom"])

El metodo find() nos encuentra en que posicion se encuentra un substring y devuelve la primera coincidencia.

El metodo replace() sustituye substring, el primer str es el ubstr a buscar y el segundo por el que vamos a sustituir

El metodo isdecimal() nos regresa si un str es un numero o no

El metodo isalnum() regresa si todos los caractres son letras o numeros

El metodo isalpha() detecta las letras del alfabet y nos devuelve una respuesta booleana

El metodo isspace() nos regresa una respuesta booleana si el str contiene puros espacios

El metodo capitalize() pone en mayuscula el primer char
de un str.

El metodo swapcase() invierte cada char, si estaba en miniscula ahora estara en mayuscula, si estaba en mayuscula ahora estara en miniscula.
